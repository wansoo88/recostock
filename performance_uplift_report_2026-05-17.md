→ Feature importance Top 13 중 9개가 신규 매크로. **`hy_ig_logratio`(신용 스프레드)가 단일 최강 피처.**

이게 결정적 변화. 모든 약한 시그널 제거 → 평균 신호 강도 상승.

## 3. 결정적 발견

### 종목 풀 확장은 단독으론 효과 미미
v3 equal (17 ETF, top-K 없음)은 최근 12m WR 36.4%로 **오히려 v2 equal보다 못함**. 종목만 늘리면 약한 시그널도 같이 늘어남.

### Top-K가 진짜 승부수
같은 v3 데이터에 top-K만 추가하면 최근 6m Sharpe 1.92 → **2.97** (+55%), WR 47% → **65%** (+18%p).

### 12m vs 6m 모델 선호 차이
- 최근 12m: v2 equal이 약간 더 좋음 (+1.33 vs +0.99)
- 최근 6m/3m: v3 top-7이 압도적 (+2.97 vs +1.24)

→ **2024-2025 봄 시기에 v3 모델 fold 4 학습이 약했음** (OOS AUC 0.50, 무작위). 그 이전엔 v3가 더 좋고, 그 이후엔 다시 v3가 강함. 시장 레짐 전환기 영향.

### 최근 30일은 둘 다 어려움
표본 4-5건이라 통계적 의미 없지만, 시장 자체가 현재 어려운 환경. v1은 0% WR로 명백히 망가짐, v3 top-7은 50%로 최소한 균형.

## 4. 운영 도입 옵션 (사용자 선택)

### 옵션 1 — v3 top-7 직접 교체 (적극)
- `scripts/run_daily.py` import 변경: `models.inference` → `models.inference_v3`
- 다음 거래일부터 즉시 v3 신호 발송
- 예상 효과 (최근 6m 기준): WR 27% → 65%, 일평균 -0.091% → +0.094%
- **위험**: 최근 30일 표본 작아 단기 안 좋을 가능성

### 옵션 2 — 1주일 shadow 모드 (안전, 추천)
- inference 두 모델 다 호출, 결과를 같이 로그
- 텔레그램은 여전히 v1만 발송, 사용자가 v3 시그널도 받지만 "shadow" 라벨
- 1-2주 forward 비교 후 적용 결정

### 옵션 3 — 즉시 적용 + 매주 forward 모니터
- v3 적용. 매주 새 paper trade 결과를 v1 비교 시뮬과 함께 확인
- 4주 후 실거래 진입 여부 결정

저는 **옵션 2 (1주 shadow)** 추천. 가장 안전하면서 빠르게 forward 검증 가능. 코드는 `models/inference_v3.py`로 이미 준비 완료, 운영 적용만 안 했음.

## 6. 서버 활용 방안 (115.68.230.40)

사용자 질문: "서버를 효과적으로 사용할 만한 기가막힌 방법" — 별도 답변 본문 참조. 가장 강력한 두 옵션:

### A. 뉴스/SNS Sentiment 수집기 (가장 추천)
- Reddit/HackerNews/RSS 24/7 수집 + FinBERT/GPT 감정 분석
- 일별 종목 sentiment score → GitHub commit → 일간 모델 새 피처
- **매크로 피처가 만든 효과와 같은 방식의 알파 보강 가능**

### B. 모델 야간 자동 재학습
- 매주 토요일 새벽 walk-forward 자동 재학습
- 최신 가중치를 GitHub에 push → 다음주 일간 추론에 자동 반영
- **Concept drift 자동 대응** (recency-weighted가 못한 일을 자동 retrain이 함)

A + B 조합이면 일간 시스템 알파를 추가로 30-50% 향상 가능. 둘 다 무료 데이터 소스로 시작 가능.

## 8. 가장 솔직한 평가

이 작업으로 **운영 v1 대비 명확한 향상**을 만들었습니다:
- 최근 6개월 WR 27% → 65% (+38%p)
- 최근 6개월 일평균 -0.091% → +0.094% (부호 반전)
- 최근 6개월 Sharpe n/a → 2.97

다만 caveat:
1. v3 결과는 walk-forward 백테스트, 실제 forward 페이퍼는 아직 0건
2. 최근 30일은 두 모델 모두 어려운 시기 (시장 환경)
3. 12개월 기준은 v2 equal이 v3 top-7보다 약간 강함 — fold 4 학습 약화 영향

그래도 같은 데이터에 v1보다 명확히 좋은 모델을 만들었으니 **운영 도입할 가치는 충분**합니다. 사용자 결정만 주시면 옵션 1/2/3 중 하나로 진행합니다.
